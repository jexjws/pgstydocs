## 开源免费：用好PostgreSQL [#open-source]

> 开源不等于免费，但用好了肯定更省钱\
> 解决云数据库和DBA成本问题

开源不等于免费，有效使用可能代价昂贵。不同 PostgreSQL 托管解决方案及其成本比较：

- Oracle 许可证 + 支持费用：约 **$3,878** / 年·vCPU
- AWS RDS PG 费用：约 **$1,920 ~ $2,640** / 年·vCPU
- Pigsty 开源版本免费，硬件成本约 **$27**/年·vCPU

## 财务降本：逃离RDS杀猪盘 [#save-more]

> 相比云数据库服务节省 50% ～ 95%+ 的费用\
> 从云数据库到本地自建成本大幅下降

云数据库杀猪盘的真相：

- RDS 成本可达 **$2,000-$2,600**/年·vCPU
- 在云 EC2 上自托管，保持弹性并节省 RDS 成本
