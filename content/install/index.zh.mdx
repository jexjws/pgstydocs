## 快速开始 [#quick-start]

[**快速开始**](/zh/install/start)：[准备](/zh/prepare) 一台具有 [SSH访问权限](/zh/prepare/admin#check-accessibility) 的 [节点](/zh/prepare/hardware#node)，新安装 [Linux](/zh/prepare/linux)系统，
使用具有免密 [`ssh`](/zh/prepare/admin#ssh) 和 [`sudo`](/zh/prepare/admin#sudo-privileges) 权限的[用户](/zh/prepare/admin#user)运行：

<Steps>
    <Step>
    使用以下命令[**下载**](/zh/prepare/software#pigsty) pigsty：

    ```bash
    curl -fsSL https://repo.pigsty.io/get | bash; cd ~/pigsty;
    ```
    </Step>

    <Step>

    执行 [`configure`](/zh/config/configure) 生成 [配置清单](/zh/config/inventory)。（<span className={"text-red-500 font-black"}>务必修改</span> 生成的 [`pigsty.yml`](/zh/config/inventory) 中的密码）

    ```bash
    ./configure
    ```

    </Step>
    <Step>

    一键 [**安装部署**](/zh/install/start) 所有组件：

    ```bash
    ./install.yml
    ```
    </Step>


</Steps>


Pigsty 安装完成！请探索 [**用户界面**](/zh/install/interface)，访问 `5432` 端口上的 [Postgres](/zh/pgsql) [服务](/zh/pgsql/service)，或 `3000` 端口上的 [Grafana](/zh/install/interface#grafana) 监控大盘（`admin` / `pigsty`）。
